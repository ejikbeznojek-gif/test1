Документ предназначен для проведения инвентаризации складов.

Порядок работы:

1. Создать документ.

2. Заполнить необходимые поля.

Склад - склад по которому проводится инвентаризация

Склад временного хранения - склад на который временно переносятся остатки  с основного склад

Статья затрат - статья затрат на которую попадает результат инвентаризации

3. Заполнить табличную часть документа. Колонки:

Номенклатура

Количество расч. - количество которое числится на складе по данным программы - возможно заполнение текущими остатками по кнопке "Заполнить остатками"

Количество факт. - фактическое количество обнаруженное на складе - колонка заполняется либо сканированием в накладную, либо сканирование в листы набора и  последующим заполнением по кнопке "Заполнить по ЛН"

4. После заполнения колонок "Номенклатура","Колличество расч." и "Количество факт." - необходимо установить состояние накладной "Отгружена" и провести документ

 

Если накладная проводится в состоянии "В работе" и  "Сканирование" - товар по накладной перемещается на склад указанный в поле "Склад временного хранения" в соотвестсвии с колонкой "Отклонение" 

Если накладная имеет  состояние "Отгружена"  - корректируются только остатки по основному складу.

 

В соучае невозможности автоматического определения цены по которой нужно оприходовать излишки в поле "Цена оприходования" необходимо проставить цену оприходования. Данный пункт выполняется только в офисе.
