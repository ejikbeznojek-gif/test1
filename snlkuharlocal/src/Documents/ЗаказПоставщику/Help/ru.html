Распределение по партиям:

Если на вкладке Доп.реквизиты установить галку -"Отгрузка с транзитного склада" и выбрать в качестве склада склад с типом Внешний склад,

тогда станет доступна вкладка "партии". При нажатии на кнопку "Распределить партии" программа попытается автоматически распределить требуемое количество по контейнерам.

Сиреневым светом выделены неиспользуемые партии (с них не взято ничего). При клике на строку с позицией, которую не удалось распределить в правой части экрана появится красная строка с нужным (запрошенным ) количеством и распределенным. В этом случае можно вручную поправить распределение по партиям, либо изменить запрошенное количество в таб. части товары. Пока запрошенное количество не сравняется с распределенным- документ не может быть проведен...

ЦЕНЫ:

При распределении по парнтиям одновременно заполняются цены в таб. части товары. Проставляется СРЕДНЯЯ цена по списаннымм партиям.
