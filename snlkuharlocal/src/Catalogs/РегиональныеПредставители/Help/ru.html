Справочник создается для отчетов которые должны формироваться в автоматическом режиме и отправляться на электронную почту

Важно

  1. Добавлять можно встроенные отчеты сделаные либо с использованием СКД, либо с использованием построителя отчета
  2. Процедуры НастроитьПостроитель (при использовании построителя отчета), Сформировать должны быть доступны из других программных модулей.
  3. В карточке физического лица должен быть заполнен хотя бы один адрес электроной почты
  4. В общих настройках подключаемого отчета должна быть настройка для конкретного получателя
