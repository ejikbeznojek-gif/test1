Обработка «Выгрузка данных в терминал сбора данных»

Данная обработка предназначена для выгрузки данных в терминал. Порядок работы с терминалами сбора данных следующий:

  • Выгрузка списка товаров в терминал
  • Работа с терминалом сбора данных
  • Загрузка результатов работы в документы

Для выгрузки списка товаров необходимо выполнить следующую последовательность действий:

  • В случае, если подключено несколько терминалов сбора данных, выбрать тот, выгрузка данных в который предполагается
  • В случае необходимости, заполнить параметры отбора товаров, подлежащих выгрузке. Сделать это можно на панели «Отбор», видимость которой задаётся при помощи кнопки «Отбор» верхней командной панели
  • После нажатия кнопки «Заполнить» происходит заполнение таблицы товаров, подлежащих выгрузке
  • Отметить строки, подлежащие выгрузке в терминал
  • Нажать кнопку «Выгрузить»
