Данная обработка предназначена для сверки остатков между удаленными подразделениями.

Порядок работы:

1. Выгрузка остатков

1.1 В форме настроек указать каталог обмена в который будет выгружен файл с остатками

1.2 Указать склады по которым будут выгружены остатки

1.3 Указать период на который будут сформированы остатки (Дата1=Дата2=Дата получения остатков)

1.4 Нажать на кнопку "Выгрузить остатки"

1.5 Наименование файла остатков имеет вид "Остатки_<Имя_склада>_<Дата>.zip", где <Имя_склада>- наименование первого склада в списке выбранных складов, <Дата>- дата получения остаков в формате "ГГГГММДД".

1.6 Полученный файл остатков необходимо отправить в офис по электронной почте.

2. Сравнение остатков

2.1. Получить из удаленного подразделения файл остатков

2.2. Сохранить файл в произвольный каталог на диске

2.3. В форме настроек указать в поле "Имя файла остатокв: " - полученный на шаге 2.2 файл остатков

2.4 Нажать на кнопку "Сравнить остатки" 

2.5 Никаких дополнительных настроек кроме указания файла с остатками не требуется !!!
